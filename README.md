# IPR Education Game

An interactive educational game designed to teach school students about Intellectual Property Rights (IPRs) including Patents, Trademarks, Copyrights, and Designs.

## Game Overview

This game is developed as part of the CIPAM initiative to create educational tools for IP awareness at the school level. The game uses gamification to make learning about intellectual property engaging and interactive.

## Current Features (MVP)

1. **Interest Selection Screen**: Students choose from three interests - Cars, Digital Gadgets, and Films
2. **Trademark Recognition Activity**: 
   - Focuses on car logos and brand identification
   - Presents images with intentional trademark mismatches
   - Provides immediate visual feedback (green for correct, red for incorrect)
   - Offers educational explanations about trademarks

## How to Play

1. Open `index.html` in a web browser
2. Select "Cars" from the interest options (other options will be available in future versions)
3. View the image and read the question: "What is wrong with this image?"
4. Choose from the two options:
   - "Image is fine"
   - "Car logo and car model is not matching"
5. Receive immediate visual feedback:
   - Green background for correct answers
   - Red background for incorrect answers
6. Read the educational explanation that appears after answering

## Technical Implementation

- **HTML5** for structure
- **CSS3** for styling and animations
- **Vanilla JavaScript** for interactivity
- Fully responsive design that works on desktop and mobile devices

## Learning Outcomes

Upon completing the game, students will:
- Understand the basic concepts of intellectual property rights
- Be able to identify trademarks and recognize trademark infringement
- Gain awareness of the importance of protecting intellectual property
- Develop critical thinking skills for recognizing IP violations
